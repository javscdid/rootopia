❓ FAQ RooTopia

Kenapa tidak bisa login? +
Pastikan kamu sudah mengatur host menggunakan PowerTunnel / Virtual Host dan Growtopia tidak sedang update.
Tools mana yang cocok untuk HP saya? +
Jika Android, gunakan PowerTunnel atau Virtual Host. Jika iOS, gunakan Shadowrocket. Jika PC, gunakan Proxifier.
Apakah RooTopia legal? +
RooTopia adalah server private, bukan server resmi Ubisoft. Main dengan tanggung jawab sendiri.
Dimana saya bisa gabung komunitas? +
Kamu bisa join Discord kami untuk tanya jawab, laporan bug, dan pengumuman. Link tersedia di halaman utama.
Apa server ini butuh VPN? +
Tidak perlu VPN. Cukup gunakan tools seperti PowerTunnel atau Virtual Host untuk mengganti host Growtopia.
