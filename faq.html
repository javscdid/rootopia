❓ FAQ RooTopia

Kenapa tidak bisa login/mental ke rgt? +
Pastikan kamu sudah mengatur host menggunakan PowerTunnel / Virtual Host dengan benar. Jika masih begitu meski kamu sudah atur dengan benar, kamu bisa pergi ke halaman "Contact Us" dan join group kami. Kamu bisa chat salah satu staff kami atau chat di general.
Apakah RooTopia berbahaya? +
Tidak.Karena kami tidak memiliki niat apapun selain membuka server GTPS. Untuk berjaga jaga, pastikan kamu delete data Growtopia kamu saat ingin login ke Real Growtopia.
Gimana cara membeli role/assets? +
Gampang banget! Kamu tinggal pergi ke halaman "Contact Us" dan kamu bisa join group kami atau klik salah satu WhatsApp Owner.
Gimana cara login ke RooTopia? +
Gampang! Kamu bisa pergi ke halaman "Tutorial" dan akan dijelaskan bagaimana cara loginnya.
Apa event server ini banyak? +
Tentu saja, RooTopia memiliki berbagai event menarik yang bisa kamu ikuti, seperti SB event, Exchange event, dan lain lain. Join kedalam gamenya untuk mendapatkan lebih banyak fitur menarik!
