Cara Login dengan Virtual Host:

  1. Download aplikasi Virtual Host.
  2. Edit host dan isi IP server kamu.
  3. Simpan dan aktifkan host.
  4. Buka Growtopia.
IP: 192.168.0.1 | Port: 17091

Cara Login di PC:

  1. Edit file `hosts` di Windows: `C:\Windows\System32\drivers\etc\hosts`
  2. Tambahkan baris:
    `192.168.0.1 growtopia1.com growtopia2.com`
  3. Buka Growtopia dan login.
