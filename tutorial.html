Tutorial Login RooTopia

Ikuti panduan lengkap di bawah ini untuk bisa bermain RooTopia di semua perangkat (Android, PC, iOS).

Client Android

Untuk bermain RooTopia di Android, kamu bisa menggunakan aplikasi PowerTunnel atau Virtual Host. Berikut langkah-langkahnya:

  1. Download dan install aplikasi PowerTunnel atau Virtual Host dari link yang diberikan oleh RooTopia.
  2. Download file config RooTopia (biasanya berupa file .json atau .txt).
  3. Buka aplikasi PowerTunnel atau Virtual Host dan masukkan config tersebut.
  4. Aktifkan VPN/host-nya di aplikasi.
  5. Setelah aktif, buka game Growtopia, dan kamu akan langsung terhubung ke server RooTopia.

Catatan: Pastikan koneksi internet stabil dan aplikasi diizinkan berjalan di latar belakang.

Client PC (Windows)

Untuk pengguna Windows, kamu perlu mengubah file hosts agar bisa terhubung ke server RooTopia:

  1. Buka Notepad sebagai administrator (klik kanan > Run as administrator).
  2. Buka file C:\Windows\System32\drivers\etc\hosts.
  3. Di bagian paling bawah file tersebut, tambahkan baris berikut:
    123.456.789.10 growtopia1.com
    123.456.789.10 growtopia2.com
  4. Simpan file (Ctrl + S), lalu tutup Notepad.
  5. Buka Growtopia seperti biasa, maka kamu akan masuk ke server RooTopia.

Catatan: Pastikan antivirus atau Windows Defender tidak memblokir perubahan file hosts.

Client iOS (Surge 5)

Untuk pengguna iOS, kamu dapat menggunakan aplikasi Surge 5 yang tersedia di App Store:

  1. Buka App Store dan cari aplikasi "Surge 5", kemudian install.
  2. Buka aplikasi Surge 5 dan import config RooTopia (format .conf atau URL config).
  3. Aktifkan koneksi di aplikasi Surge 5.
  4. Setelah Surge aktif, buka Growtopia seperti biasa.

Catatan: Beberapa config mungkin membutuhkan versi premium dari Surge.
